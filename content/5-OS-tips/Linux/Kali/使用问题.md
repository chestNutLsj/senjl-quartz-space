## apt-get
```shell
# 更新软件包列表
apt-get update

# 更新软件包
apt-get upgrade

# 升级发行版系统
apt-get dist-upgrade

# 删除缓存包
apt-get clean

# 删除未安装的deb包
apt-get autoclean
```

## 汉化
```shell
# 设置地区语言选项(建议勾选en_US.UTF-8, zh_CN.UTF-8,空格勾选)
sudo dpkg-reconfigure locales

# 解决编码错误显示问题
sudo apt-get install ttf-wqy-microhei ttf-wqy-zenhei xfonts-wqy

# 重启即可
sudo reboot
```
